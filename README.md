# Cold Lovers

A dark, moody short story blog for JustUtter February.

## Deploy to Vercel

1. Push this repo to GitHub
2. Connect to Vercel
3. Add custom domain: `coldlovers.justutter.com`

## Files

- `index.html` - Main site with embedded images
- `cold-lovers-*.png` - Theme images (embedded as base64, files kept for reference)
